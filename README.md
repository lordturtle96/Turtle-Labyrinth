# Turtle-Labyrinth

Static website for **Sanity & Secrets**.

## Pages
- `index.html` — homepage
- `about.html` — about page
- `blog.html` — Blog Bog index page
- `posts/first-post.html` — opening personal post
- `posts/post-template.html` — duplicate this when creating a new post
- `paid_tier.html` — supporter offer page
- `welcome_email.html` — welcome message page
- `styles.css` — shared site styling

## How to add a post
1. Copy `posts/post-template.html`
2. Rename the file
3. Edit the title, date, description, and body
4. Add a link to the new post in `blog.html`

## Deployment
This repository is suitable for GitHub Pages hosting.
