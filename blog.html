Easy posting workflow

How to add a new post later

  1. Open posts/post-template.html
  2. Duplicate it and rename it
  3. Edit the title, date, excerpt, and content
  4. Add a new entry here on blog.html

Visitors can read your posts, but only you or other repository collaborators can change them.
